# App Store Connect - Promotional Text Options

## About Promotional Text
- **Maximum length:** 170 characters
- **Can be updated:** Anytime without app review
- **Appears:** Above the app description on the App Store
- **Purpose:** Highlight key features and entice users to download

## Tips for Choosing

1. **Test different messages:** You can update promotional text anytime without review
2. **A/B test:** Try different options and see which performs better
3. **Seasonal updates:** Change for holidays or special events
4. **Feature updates:** Update when adding new major features
5. **Keep it fresh:** Update every few months to maintain visibility

# Icon Assets Directory

Place your 1024x1024 PNG icon file here as `icon.png`.

The icon should:
- Be 1024x1024 pixels
- Have a square aspect ratio
- Use PNG format
- Follow the design guidelines in `ICON_DESIGN.md`

After placing the icon, run:
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

This will automatically generate all required icon sizes for iOS and Android.

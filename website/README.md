# Nyx Support Website

This folder contains the support website files for Nyx.

## Files

- `index.html` - Main support page with FAQ and contact information
- `GITHUB_PAGES_SETUP.md` - Step-by-step guide to host on GitHub Pages

## Quick Start

1. Read `GITHUB_PAGES_SETUP.md` for detailed instructions
2. Create a GitHub repository (name it `nyx-support`)
3. Upload `index.html` to the repository
4. Enable GitHub Pages in repository settings
5. Your website will be at `https://YOUR_USERNAME.github.io/nyx-support/`

## Customization

Edit `index.html` to customize:
- Email address
- FAQ content
- Styling (colors, fonts, layout)
- Additional sections

The page is styled to match Nyx's dark theme (#0E0E11 background, #2EE6A6 accent color).

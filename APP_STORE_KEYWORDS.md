# App Store Connect - Keywords Options

## About App Store Keywords
- **Maximum length:** 100 characters total (including commas)
- **Format:** Comma-separated, no spaces after commas
- **Cannot include:** Words already in your app name or subtitle
- **Purpose:** Help users find your app when searching
- **Strategy:** Use single words or short phrases users might search for

## Common Mistakes to Avoid

❌ **Don't include:**
- App name (Nyx) - Already indexed
- Subtitle words (Secure, Privacy, Vault) - Already indexed
- Spaces after commas
- Duplicate words
- Irrelevant keywords
- Brand names of competitors

✅ **Do include:**
- Single relevant words
- User search intent terms
- Feature names
- File types supported
- Use cases (backup, organize, protect)

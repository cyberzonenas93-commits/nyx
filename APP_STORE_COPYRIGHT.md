# App Store Connect - Copyright Information

## Copyright Field Format

For the **Copyright** section in App Store Connect, use this format:

```
© 2025 Angelo Nartey
```

or

```
Copyright 2025 Angelo Nartey
```

## Standard Format

**Format:** `© [YEAR] [COPYRIGHT HOLDER NAME]`

**Examples:**
- `© 2025 Angelo Nartey`
- `© 2025 angelonartey`
- `Copyright 2025 Angelo Nartey`

## Current Year

Since we're in 2025, use:
```
© 2025 Angelo Nartey
```

## If Using a Company Name

If you have a company/business name, use:
```
© 2025 [Your Company Name]
```

## What to Enter in App Store Connect

**Copyright field:**
```
© 2025 Angelo Nartey
```

**Note:** The copyright symbol (©) is typically automatically added by App Store Connect, so you can also just enter:
```
2025 Angelo Nartey
```

But using the full format `© 2025 Angelo Nartey` is safest.

## Additional Notes

- Use the current year (2025)
- Use your legal name or business name
- This appears in the App Store listing
- Should match your developer account information
